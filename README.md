# kahia-pointage 📍

Système de pointage GPS via WhatsApp → n8n → Supabase.

## 🔗 Liens utiles
- n8n : https://votre-url-railway.up.railway.app
- Supabase : https://supabase.com/dashboard

## ✅ Fonctionnalités

- Dashboard HTML temps réel
